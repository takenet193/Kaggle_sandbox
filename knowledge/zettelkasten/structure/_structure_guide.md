# Structure（構造ノート）

複数のノートをまとめるハブノートを保存する場所です。

## 用途

- 特定トピックに関するノートを束ねる
- 思考の流れを整理する
- プロジェクト単位でノートをグループ化

## 命名規則

```
topic_structure_YYYYMMDDHHMMSS.md
```

例：`feature_engineering_structure_20251228140000.md`

## ノート構造

```markdown
---
id: YYYYMMDDHHMMSS
title: トピック名 - 構造ノート
type: structure
tags: [structure, topic]
created: YYYY-MM-DD
---

# トピック名

## 概要
このトピックについての全体像

## 主要なノート

### 基礎
- [[基礎ノート1|説明]]
- [[基礎ノート2|説明]]

### 応用
- [[応用ノート1|説明]]
- [[応用ノート2|説明]]

### 実験・実践
- [[実験ノート1|説明]]

## 未解決の問い
- まだ分かっていないこと
- 今後調べたいこと

## 関連する構造ノート
- [[別の構造ノート]]
```

## 特徴

- 構造ノートは「地図」のような役割
- 個々のノートは詳細、構造ノートは全体像
- グラフビューで中心的なハブになる
