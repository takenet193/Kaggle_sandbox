# 引用: ノートの品質はリンク網から生まれる（ルーマン）

## 引用（原文）

> “Every note is only an element which receives its quality only from the network of links and back-links within the system. A note that is not connected to this network will get lost in the card file and will be forgotten by it. Its rediscovery depends on accidents and on the vagary that this rediscovery means something at the time it is found.”

- [[luhmann_communicating_with_slip_boxes_20260104|Communicating with Slip Boxes（ルーマン）]]

## メモ

- ノートの価値は「中身」だけでなく、リンク/バックリンクのネットワーク内での位置づけから生まれる
- 未接続のノートは“沈む”／再発見は偶然性に依存する → 運用規律（リンク整備・索引・構造ノート）が重要
