# Index（インデックスノート）

トピック別の入口となるインデックスを保存する場所です。

## 用途

- アルファベット順・五十音順のインデックス
- タグ別のノート一覧
- プロジェクト別のエントリーポイント

## 構造ノートとの違い

| 構造ノート | インデックスノート |
|------------|-------------------|
| 思考の流れを整理 | 単純なリスト・目次 |
| 説明・コンテキストあり | リンクの羅列 |
| 深い階層構造 | フラットな一覧 |

## 命名規則

```
index_カテゴリ名.md
```

例：`index_models.md`, `index_experiments.md`

## ノート構造

```markdown
---
id: YYYYMMDDHHMMSS
title: インデックス - カテゴリ名
type: index
tags: [index]
created: YYYY-MM-DD
updated: YYYY-MM-DD
---

# インデックス: カテゴリ名

## A
- [[apple_20251228|Apple]]

## B
- [[banana_20251228|Banana]]

## K
- [[kaggle_basics_20240101123045|Kaggle基本戦略]]

...
```

## 運用

- 必要に応じて作成（最初から作らなくてもOK）
- ノートが増えてきたら整理用に追加
- 自動生成スクリプトで更新も可能
