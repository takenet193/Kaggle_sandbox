# Permanent（永続ノート）

自分の言葉で書かれた、永続的な知識ノートを保存する場所です。

## 特徴

- 一時的なメモではなく、長期的に価値のある知識
- 自分の考えや理解が含まれている
- 他のノートとリンクで繋がっている

## 文献ノート（references/）との違い

| Permanent | References |
|-----------|------------|
| 自分の言葉で書く | 出典の要約・引用 |
| 複数の情報源を統合 | 単一の情報源に対応 |
| 長期的に価値がある | 参照用 |

## 命名規則

```
description_YYYYMMDDHHMMSS.md
description_YYYYMMDDHHMMSS_a.md  ← 同一秒で2つ目（衝突回避。通常は不要）
```
